This is the Audio library repository

Steps to get the player working are as follows.

Step 1 - run the create_tables.py file which will create a sqlite db for your songs.

Step 2 - run the song_api.py.

Step 3 - While running the song_api.py file, open and run the main_app_controller.py file.
	 This will open the library GUI and press help to know more on how to use the player.
